Configuração da API do YouTube
Para capturar automaticamente o título dos vídeos do YouTube, você precisa configurar a API do YouTube:
Como obter uma chave da API do YouTube:
  1. Acesse o Google Cloud Console
  2. Crie um novo projeto ou selecione um existente
  3. Vá para "APIs e Serviços" > "Biblioteca"
  4. Procure por "YouTube Data API v3" e ative-a
  5. Vá para "APIs e Serviços" > "Credenciais"
  6. Clique em "+ CRIAR CREDENCIAIS" > "Chave de API"
  7. Copie a chave gerada e cole no campo acima
  8. Recomendado: Clique em "EDITAR CHAVE DE API" e restrinja o uso
